# Screenshots

Place your screenshots in the appropriate folders below.

---

## Folder Structure

```
screenshots/
├── app-ui/                  ← Application UI screenshots
│   ├── home.png
│   ├── products.png
│   ├── cart.png
│   ├── checkout.png
│   └── payment.png
│
├── jenkins-pipeline/        ← Jenkins CI/CD screenshots
│   ├── pipeline-success.png
│   ├── pipeline-stages.png
│   └── build-history.png
│
├── aws-infrastructure/      ← AWS Console screenshots
│   ├── ec2-instances.png
│   ├── vpc.png
│   ├── alb.png
│   ├── target-group.png
│   └── security-groups.png
│
└── kiro-ai/                 ← Kiro AI session screenshots
    └── kiro-session.png
```

---

## Naming Convention

Use lowercase with hyphens:
- `home-page.png` not `HomePage.PNG`
- `jenkins-success.png` not `Jenkins Success.png`
- `ec2-dashboard.png` not `ec2dashboard.png`
